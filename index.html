How to Install Ollama

1 Install Ollama

Download and install the Ollama application for your operating system.

https://ollama.com/download

2 Download a Model

Open your terminal or command prompt and run one of the following commands to download a model. The model will run automatically when SparkVoice needs it.

Small & Fast (Recommended for < 8GB VRAM) ~2GB
ollama run qwen2.5:3b-instruct-q4_k_m
Large & Accurate (Requires 8GB+ VRAM) ~9GB
ollama run qwen2.5:14b-instruct-q4_k_m
